# 数组

## 一、理论基础

### 1.1 数组

**数组是存放在连续内存空间上的相同类型数据的集合。**
1. 数组下标都是从0开始的；
2. 数组内存空间的地址是连续的。

### 1.2 哈希表

哈希表（hash table），又称散列表，它通过**建立键 key 与值 value 之间的映射**，实现高效的元素查询。具体而言，我们向哈希表中输入一个键 key ，则可以在 
 时间内获取对应的值 value 。

 --> 用key查询value

- 在哈希表中进行**增删改查**的时间复杂度都是O(1)

```java
// 初始化
Map<Integer, String> map = new HashMap<>();

// 添加操作 -- 哈希表中添加键值对(key, value)
map.put(12836, "小哈");

// 查询操作 -- 向哈希表中输入键key，得到值value
String name = map.get(12836);

// 删除操作
map.remove(12836)
```

- 哈希表有三种常用遍历方式：**遍历键值对，遍历键，遍历值**

```java
// 遍历键值对
for(Map.Entry <Integer, String> kv : map.entrySet()){
    System.out.println(kv.getKey() + " -> " + kv.getValue());
}

// 遍历键key
for(int key : map.keySet()){
    System.out.println(key);
}

// 遍历值
for(String val : map.values()){
    System.out.println(val);
}
```

## 二、练习

### 2.1 两数之和

原题链接：[1. 两数之和](https://leetcode.cn/problems/two-sum/description/?envType=study-plan-v2&envId=top-100-liked)

题解来源：[哈希表题解](https://leetcode.cn/problems/two-sum/solutions/6873/jie-suan-fa-1-liang-shu-zhi-he-by-guanpengchn/?envType=study-plan-v2&envId=top-100-liked)

题目要求：给定一个整数数组 `nums` 和一个整数目标值 `target`，请你在该数组中找出 和为目标值 `target` 的那 两个 整数，并返回它们的数组下标。

思路：

1. 暴力枚举，两个for循环
  
  - 时间复杂度：O(n^2)
  - 空间复杂度：O(1)

```java
class Solution {
    public int[] twoSum(int[] nums, int target) {
        for(int i = 0; ; i++){
            for(int j = i + 1; j < nums.length; j++){
                if(nums[i] + nums[j] == target){
                    return new int[]{i, j};
                }
            }
        }
    }
}
```

2. 哈希表
   
   利用哈希容器map来降低时间复杂度

   --> 从左往右遍历数组nums，i为当前下标，对于每个值判断map是否存在`target-nums[i]` 的key值

   如果存在则找到了两个值，不存在就将当前的 `(nums[i], i)` 存到map中，并继续向下遍历直到找到为止

   如果最终都没有结果，就抛出异常

   - 时间复杂度：O(n)
   - 空间复杂度：O(n)

```java
class Solution {
    public int[] twoSum(int[] nums, int target) {
        Map<Integer, Integer> map = new HashMap<>();

        for(int i = 0; i < nums.length; i++){

            if(map.containsKey(target - nums[i])){
                return new int[] {map.get(target-nums[i]), i};
            }

            map.put(nums[i], i);
        }

        throw new IllegalArgumentException("No two sum solution");
    }
}
```

### 2.2 字母异位词分组***

原题链接：[49.字母异位词分组](https://leetcode.cn/problems/group-anagrams/?envType=study-plan-v2&envId=top-100-liked)

要求：

**字母异位词：** 如果两个字符串从小到大排序后相等，那这两个字符串就互为字母异位词。

给一个字符串数组，要把字母相同但排列不同的字符串（异位词）放在一起。

输入：`["eat", "tea", "tan", "ate", "nat", "bat"]`
输出：[["bat"],["nat","tan"],["ate","eat","tea"]]

思路：

哈希表进行分组，把排序后的字符串当作哈希表的 key，排序前的字符串加到对应的列表中（哈希表的 value）。

```java
class Solution {
    // List<List<String>> 是一个列表，每个元素又是一个列表 -- 即分好组的单词
    public List<List<String>> groupAnagrams(String[] strs) {

        // key是排序后的字符串，value是原始字符串列表
        Map<String, List<String>> map = new HashMap<>();

        // 遍历输入的每个字符串
        for(String s : strs){

            // 把s排序，作为哈希表的key
            char[] sortdS = s.toCharArray(); // 将字符串转换为字符数组
            Arrays.sort(sortdS);

            // 排序后相同的字符串分到同一组 -- 将排序后的字符数组转为字符串作为key
            // computeIfAbsent如果key不存在，执行后面的函数创建新的ArrayList
            // 如果key存在，直接返回已存在的list，然后将当前字符串s添加到这个list中
            map.computeIfAbsent(new String(sortdS), _-> new ArrayList<>() ).add(s);

            // 上面的一行等价于
        //     String key = new String(sortdS);
        //     if(!map.containsKey(key)){
        //         map.put(key, new ArrayList<>());
        //     }
        //     map.get(key).add(s);
        // }

        // 哈希表的value保存分组后的结果
        return new ArrayList<>(map.values());
    }
}
```

### 2.3 最长连续序列

原题链接：[128.最长连续序列](https://leetcode.cn/problems/longest-consecutive-sequence/description/?envType=study-plan-v2&envId=top-100-liked)

难点：找到连续序列的起点x --> 思路-如果一个数的前一个数不存在于数组中，那么这个数肯定是某个序列的起点

```java
class Solution {
    public int longestConsecutive(int[] nums) {
        Set<Integer> st = new HashSet<>();
        for(int num : nums){
            st.add(num); // 把nums的数都放进一个哈希集合中，保证在O(1)判断数字是否在nums中
        }

        int ans = 0;
        for(int x : st){

            if(st.contains(x-1)){  // 遍历，当x不是序列起点
                continue;
            }

            // 找到了序列起点x
            int y = x + 1;
            while(st.contains(y)){ // 查找x的下一个数是否在数组中
                y++;
            }

            // 循环结束后，y-1是最后一个在哈希集合中的数 --> 当y不存在于集合中时会退出循环，但此时y已经+1，因此y-1是最后一个在哈希集合中的数
            ans = Math.max(ans, y -x); // 从x到y-1共有y-x个数
        }

        return ans;

    }
}
```
