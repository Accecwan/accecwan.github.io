### 1.2 数组模拟栈

栈：**先进后出**

- push：入栈，栈顶添加元素

- pop：出栈，栈顶移除元素

```java
int[] stack = new int[10];  // 创建一个数组作为栈
int top = 0;                 // 栈顶指针，指向下一个要存放的位置

// 入栈操作
stack[top] = value;  // 在top位置放入元素
top++;               // 栈顶指针上移

// 或者简写为：
stack[top++] = value;  // 先赋值，再移动指针
```

## 二、练题

### 2.1 移动零

原题链接：[283.移动零](https://leetcode.cn/problems/move-zeroes/description/?envType=study-plan-v2&envId=top-100-liked)

题解来源：[原地栈+交换元素](https://leetcode.cn/problems/move-zeroes/solutions/2969353/kuai-man-zhi-zhen-wei-shi-yao-ke-yi-ba-s-1h8x/?envType=study-plan-v2&envId=top-100-liked)

要求：给定一个数组 `nums`，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。

思路：

两种方法

1. **原地栈**：用一个栈记录非零元素，非零入栈，并记录有几个零，最后在末尾加上零
   
    - 为了做到O(1)空间复杂度，直接用nums作为栈
    - 入栈就是 让nums[stackSize] = nums[i], 然后把stackSize+1

```java
演示

数组 0 1 0 3 12
索引 0 1 2 3 4
StackSize == 0

第一步 -- 0不入栈
数组 0 1 0 3 12
索引 0 1 2 3 4
stackSize == 0

第二步 -- 1入栈
数组 1 1 0 3 12
索引 0 1 2 3 4
StackSize == 1

第四部 -- 3入栈
数组 1 3 0 3 12
索引 0 1 2 3 4
StackSize == 2
```

2. **双指针+交换元素**：

从左到右遍历`nums[i]`，同时维护另一个下标i_0（初始为0），并保证下标区间[i_0, i - 1]都是空位，且i_0指向最左边的空位。

当nums[i]!=0，把nums[i]移动到最左边的空位上，也就是交换nums[i]和nums[i_0]，交换后把i_0和i都加1，从而使【[i_0 , i − 1] 都是空位】这一性质仍然成立。

如果nums[i] = 0，无需交换，把 i + 1 即可

#### 方法一：原地栈

```java
class Solution {
    public void moveZeroes(int[] nums) {

        int stackSize = 0;
        for(int i : nums){
            if(i != 0){
                nums[stackSize] = i; // 把i入栈
                stackSize++;
            }
        }

        // 把剩余位置全部填0
        Arrays.fill(nums, stackSize, nums.length, 0);
        
    }
}
```

- 时间复杂度：O(n)
- 空间复杂度：O(1)

#### 方法二：双指针+交换元素

```java
class Solution {
    public void moveZeroes(int[] nums) {

        int i0 = 0;

        for(int i = 0; i < nums.length; i++){
            if(nums[i] != 0){
                // 不等于0， 交换i_0和i，并且两个都要+1
                int tmp = nums[i];
                nums[i] = nums[i0];
                nums[i0] = tmp;
                i0++;
            }
        }
    }
}
```

- 时间复杂度：O(n)
- 空间复杂度：O(1)
